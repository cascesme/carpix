StorageService public API (from RESEARCH.md "Complete StorageService skeleton"):
  __init__(self, base_dir: Path) -> None
    self._base = base_dir.resolve()  # resolve symlinks on base itself

  async def save(self, brand_key: str, model_key: str, year: int, data: bytes) -> Path
    target_dir = anyio.Path(self._base) / brand_key / model_key / str(year)
    await target_dir.mkdir(parents=True, exist_ok=True)
    target_file = target_dir / "image.jpg"
    await target_file.write_bytes(data)
    return Path(target_file)   # return pathlib.Path, not anyio.Path

  def _validated_path(self, brand_key: str, model_key: str, year: int) -> Path
    candidate = self._base / brand_key / model_key / str(year) / "image.jpg"
    resolved = candidate.resolve()
    if not resolved.is_relative_to(self._base):   # self._base already resolved in __init__
        raise ValueError(f"Path traversal attempt: {candidate!r}")
    return resolved

  def file_response(self, brand_key: str, model_key: str, year: int) -> FileResponse
    path = self._validated_path(brand_key, model_key, year)
    return FileResponse(path=path, media_type="image/jpeg")

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| caller → StorageService.save() | brand_key, model_key, year inputs arrive from Phase 5 ImageService (pre-normalized by canonical_key); StorageService must be safe even with non-normalized inputs |
| caller → StorageService.file_response() | brand_key, model_key, year inputs; must be validated via path guard before FileResponse is constructed |
| filesystem → FileResponse | Resolved path must be inside BASE_IMAGES_DIR; symlink escape must be blocked |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | StorageService._validated_path | mitigate | `candidate.resolve().is_relative_to(self._base)` — stdlib method collapses `..` and follows symlinks before comparison; self._base resolved in __init__ to handle symlinked Docker volumes |
| T-02-02 | Information Disclosure | StorageService.file_response | mitigate | Same guard as T-02-01: resolved path outside base raises ValueError before FileResponse construction; router converts to HTTP 400, not 500 |
| T-02-03 | Tampering | path construction with int year | mitigate | `str(year)` enforced in path construction; year comes in as int from FastAPI router (type-safe); canonical_key strips all non-alphanumeric from brand/model inputs |
| T-02-04 | Tampering | null byte injection via brand_key/model_key | accept | canonical_key() (Phase 1) strips all non-alphanumeric characters, eliminating null bytes; StorageService guard provides defense-in-depth even without normalization |
| T-02-SC | Tampering | pip/cargo installs | accept | No new packages installed in Phase 2 — anyio and starlette are existing transitive deps |
